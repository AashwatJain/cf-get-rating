# Get Rating — Codeforces Extension

Chrome extension that shows problem rating and tags on Codeforces problem pages.

## Features

- Shows problem rating in the sidebar (always visible)
- Toggle button to show/hide problem tags
- Contest Standings link

## Installation

1. Clone the repository
   ```bash
   git clone https://github.com/AashwatJain/cf-get-rating.git
   ```
2. Go to `chrome://extensions` in your browser
3. Turn on **Developer mode** (top-right corner)
4. Click **Load unpacked** and select the cloned `cf-get-rating` folder
5. Visit any Codeforces problem page — the extension will work automatically
